# Number guessing game on base network

## How to play

. User are expected to sign-in using their wallet that supports base network
. There's an input field used to collect users guess(a number)
. User clicks the check button to know if the guess is right or not, a feedback is given after each try.
. The reset game is used to reset the game back to it's initial state

### Features 
1. The game play
2. Display of leaderboard
